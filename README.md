
# HJL-Notepad 后端部分


## 简介
这是HJL-Notepad记事本应用的后端部分，使用Node.js + MongoDB进行搭建，Restful Service,为前端提供登录，增删改查等api接口。

##使用

```
$ git clone https://github.com/HUJINLIANG/HJL-Notepad-BE.git
$ cd HJL-Notepad-BE
$ npm install
```

## dev
```
node app.js
```

## prod
```
  npm start
```

## 布署

使用pm2部署到服务器上，然后配置Nginx
